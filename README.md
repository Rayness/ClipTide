## General Information: 
> [!NOTE]
> This small program is designed to download video content in audio and video formats from YouTube video hosting. It all runs in Python using the [ydl](https://github.com/ytdl-org/youtube-dl) bibilio for downloading videos, and [rich]() for the console interface.

> [!IMPORTANT] 
> ### Please note!
> The application requires **[Python](https://www.python.org/downloads/)** installed. If it is not, an automatic installation will be performed. By downloading the applications you agree to the installation of **Python** on your personal computer.
